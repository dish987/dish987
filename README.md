### 📂 Projetos em Destaque

#### [[☕ Villa Do Café | Full-Stack Menu & Inventory]](https://github.com/dish987/Site-Cafe-)
Um sistema completo para cafeterias que une um cardápio digital dinâmico a um painel administrativo para gestão de estoque.
* **Destaque:** Implementação de um servidor **Node.js/Express** integrado ao **SQLite3** para persistência de dados.
* **Funcionalidades:** Dashboard administrativo (CRUD completo), upload de imagens de produtos via **Multer**, controle de estoque em tempo real e cálculo automático de margem de lucro.
* **Lógica:** O sistema marca automaticamente itens como "esgotados" no front-end quando o estoque chega a zero.

#### [🧪 Rick and Morty Explorer (API Integration)](https://github.com/dish987/API-Rick-And-Morty)
Aplicação interativa que consome a API oficial de Rick and Morty para busca e filtragem dinâmica de personagens.
* **Foco:** Uso avançado de **JavaScript Assíncrono (Async/Await)** e tratamento de erros para garantir uma experiência de usuário sem falhas.

#### [🗓️ Digital Task Planner](https://github.com/dish987/To-Do-List)
Aplicação de planejamento diário focada em modularidade.
* **Foco:** Desenvolvido inteiramente com **Programação Orientada a Objetos (POO)**, garantindo um código limpo e organizado.
